## This Post Includes

This blog system features:

1. **Responsive Images** - Images are automatically optimized across different screen sizes
2. **MDX Support** - Write posts using Markdown with embedded React components
3. **Automatic Cache Busting** - Images are hashed for efficient caching
4. **Tags & Metadata** - Organize and categorize your posts

## Creating Sample Images

To add your own images:

1. Save image files to `public/images/blogs/` (PNG, JPG, SVG, WebP)
2. Reference in frontmatter: `featuredImage: /images/blogs/your-image.jpg`
3. Inline markdown: `![Alt text](/images/blogs/your-image.jpg)`
4. Images are automatically:
   - Optimized to multiple sizes (480px, 768px, 1200px)
   - Given unique hashes for cache busting
   - Lazy-loaded for performance
   - Made responsive with srcset

## Code Example
